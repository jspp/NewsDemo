# Deatail

#### 介绍
Java8 的一些新特性介绍和示例。

包含：

- 函数式接口
- 接口新特性
- Lambda表达式
- 方法引用
- Optional
- Stream 流
- ForkJoin 
- json-lib : Json 解析 序列化 反序列化
- gson : Json 解析 序列化 反序列化
- lambok : 简化插件
- jsoup : HTML解析 常用于爬虫或者数据抓取
- WebFlux SpringBoot Rest
- mongoDB  CURD
- SpringJDBC ORM  CURD
- Xstream 解析复杂XML to Bean


说明：

- 项目基于Java8
- 编辑器 IDEA
- 字符编码 UTF-8
